Instruction Manual
How to use the encyclopedia modules and instant search.
1. Choose the language from the top language selector. 2. Open Mudrā Encyclopedia, Shloka Encyclopedia, or Metaphor Encyclopedia. 3. Type in the search box — results will come immediately while typing. 4. You can search by mudra name, shloka word, beejakshara, philosophical idea, or metaphorical phrase. 5. Use Print, Save, or Share to take the visible result in the selected language mode. 6. The current language layer is a structured display mode and can be expanded further later.
