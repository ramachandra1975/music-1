Instruction Manual
How to use the stable merged build.
1. Use the tabs to move between Rāga Finder, Creator, Mudrā Encyclopedia, Sacred Search, Metaphors, and Recording. 2. In Sacred Search, type terms like Rudram, Namakam, Chamakam, Tat Tvam Asi, Hreem, Lalita, or Gayatri — results appear immediately. 3. In Recording & Sound, you can play a test tone and, if the browser allows microphone access, record, stop, play, and download your voice. 4. Use the language selector for structured output mode and the Print / Save / Share buttons for the current visible view. 5. This is a stable merged base, so the essential modules are restored together instead of replacing one another.
